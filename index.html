⛏️ TerraForge 遊び方

🎯 目的

地面を掘って無限鉱脈を見つけ、発電機で電力を確保しながら機械で 採掘 → 運搬 → 精錬 → 売却 の自動化ラインを作ってお金を稼ごう!(所持金 $500 からスタート)
累計収益が一定額に達すると 🏆 実績ボーナスがもらえるぞ!

📋 手順

  1. ⛏️ 掘るでタイルをタップして掘り下げる(最大3段)。鉱石が見つかる!
    🔍 探知ボタンでカメラ周辺の鉱脈の場所と深さが見える
  2. 露出した鉱石の上に 🛠️ ドリル を設置(自動採掘)。鉱脈は枯れないが、売値は控えめ
  3. ➡️ コンベアで鉱石を運ぶ。精錬炉・販売機・他のコンベアなど接続できる機械の近くに設置すると自動でその方向へ曲がって接続される。狙った向きにしたい時は「向き」ボタンで回転
  4. ⚡ 発電機を置いて電力を確保。電力不足だとドリルと精錬炉が停止する
  5. 🔥 精錬炉で鉱石をインゴットに(売値UP!)
  6. 💰 販売機に流し込むとお金に変わる

💎 鉱石と売値

⚫ 石炭(深さ1) 鉱石$1 → インゴット$4
⚙️ 鉄(深さ1) 鉱石$2 → インゴット$6
🟠 銅(深さ2) 鉱石$4 → インゴット$14
🥈 銀(深さ2) 鉱石$6 → インゴット$20
✨ 金(深さ3) 鉱石$10 → インゴット$34
💎 ダイヤ(深さ3) 鉱石$18 → インゴット$60

🔀 応用装置

🔀 分岐器:背面から来たアイテムを正面・左・右の3方向に順番に振り分ける
🔗 合流機:背面・左・右から来たアイテムを1本の出力にまとめる
📦 チェスト:アイテムを保管(最大300個)。「掘る」ツールでタップすると中身をまとめて売却できる。向いている方向に少しずつ自動排出もされる
🎯 フィルター:指定した鉱石は正面へ、不一致アイテムは左右へ分岐する3方向ソーター。下の「品目」ボタンで対象を選べる
⚡ 発電機:12電力を供給。ドリルは2、精錬炉は4電力を消費する

🏔️ 地形と接続ルール

広大なマップには丘や谷などの高低差がある(標高0〜3段)。鉱石の「深さ」はその場所の地表からの深さなので、丘の上でも谷底でも掘り方は同じ。
コンベア・分岐器・合流機・チェストなどは、地表の高さの差が1段までなら隣のマスと接続できる。2段以上離れると接続できないので、⛏️ 掘って地面を下げて高さを揃えながらラインを敷こう。

🎮 操作

タップ:ツールを使う / 1本指ドラッグ:マップ移動(指を離すと慣性でスッと止まる)
設置済みの機械を長押し:機械の上に「回転・移動・削除」ボタンを表示
ピンチ:ズーム / 右のパッド:視点回転・角度・ズーム(なめらかに動きます)
PC:マウスホバーで設置マス確認、矢印キー=視点、WASD=移動、+/-=ズーム
「掘る」ツールで機械をタップ:ドリル・精錬炉・合流機の状態を確認(チェストは中身を売却)
🔊 右上のスピーカーボタン:効果音のON/OFF
